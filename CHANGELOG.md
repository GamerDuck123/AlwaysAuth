# v0.5.0 - All the versions!
This update brings support for many different versions than the latest, in some cases all the way down to 1.8.8 up to 26.1!

I plan on expanding *all* server types to go down to 1.8.8 and the latest version, just the modded platforms take a bit longer!

Here is a full list of supported versions now:

| Version    | Support Range  |
|------------|----------------|
| Spigot     | 1.8.8 -> 26.1  |
| Paper*     | 1.8.8 -> 26.1  |
| Fabric     | 1.20 -> 26.1   |
| Neoforge** | 1.20.4 -> 26.1 |

\* With paper before 1.21.4 you can use the spigot, as paper specific plugins didn't really exist before that

\** Neoforge doesn't offer support for below 1.20.4, so I have to do some digging to find information
